React static boilerplate
========================

[React static boilerplate](https://github.com/kompot/react-static-boilerplate) boilerplate to produce static websites in a sensible way.

Getting started
---------------
1. `npm install` to install dependencies
2. `npm start` to develop with hot reloading
3. `npm run build` to produce production-ready version with everything minified in `dist` folder.
